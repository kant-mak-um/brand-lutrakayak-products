# brand-lutrakayak-products

**Brand:** LutraKayak  
**Domain:** https://lutrakayak.com (apex) and https://www.lutrakayak.com  
**Category:** Modular kayaks — river and ocean configurations

## Purpose
This repository hosts the public-facing product definitions, documentation, and marketing-ready assets for **LutraKayak**.

LutraKayak focuses on **modular kayak systems** that can be configured for multiple environments:
- river-capable builds (durability, impact resilience, easy field service)
- ocean-capable builds (stability, efficiency, saltwater-ready accessories)

## Commerce & Memberships
Sales, memberships, and checkout will be handled by **WaterWolf** (private platform).  
This repo is intentionally **public** and contains only publishable product and documentation assets.

## What’s in this repo (initial)
- `/site/` – static website content (landing pages and product pages)
- `/catalog/` – product placeholders (to be populated as kits and accessories are finalized)
- `/LICENSE` and `/LICENSES/` – licensing scaffold (per your standard)

## Status
Pre-launch. Content will expand as production ramps and LutraKayak kits/accessories are released.
